# AI Risk Prediction Model

## Data Used

The model reads `data/stock_analysis_result.csv`, which contains one calculated risk-analysis record for each tracked stock. The prediction output is saved to `data/ai_risk_prediction.csv` and copied here as an archived reporting artifact.

## Feature Variables

The model uses three numerical features:

- `total_return`: cumulative return across the analysis period
- `volatility`: standard deviation of daily returns
- `max_drawdown`: largest peak-to-trough decline during the period

Feature-importance output is available in `feature_importance.csv`.

## Model Objective

`python/ai_risk_model.py` trains a `RandomForestClassifier` to predict each record's risk category. The target variable is `risk_level` from the stock-risk analysis dataset.

## Risk-Level Classification

The upstream analysis script assigns risk levels using volatility:

| Volatility | Risk level |
| --- | --- |
| Less than 0.02 | Low |
| 0.02 to less than 0.04 | Medium |
| 0.04 or greater | High |

## Output Description

`ai_risk_prediction.csv` retains the stock identifiers, analysis metrics, original risk level, and the `AI_Risk_Prediction` column generated by the model. These results can be imported into MySQL and displayed in Power BI.

## Reproducibility

Run the following command from the project root after installing dependencies:

```bash
python python/ai_risk_model.py
```

The script uses a fixed `random_state=42` for the train/test split and Random Forest model.
